## ✨ 核心特性 / 亮点

1. **🌊 扩招“抽水效应”自动计算引擎**
   - 不再只看一分一段表吃死理。若目标排位前方的名牌院校今年全线扩招，系统会自动折算向上拉升的「等效补偿位次 (V_eff)」，精确找出“今年能被抽上去”的超级捡漏机会！

2. **⚔️ “物化同选” 惨烈赛道精准过滤**
   - 随 2024 新规，理工农医大多要求组合选科“物理+化学”。如果考生是“物生地/物生政”的“单物理”选手，系统会残酷但果断地筛掉所有受限的专业组，让你看清这条人挤人的狭窄赛道。

3. **📊 40 个平行组自动排兵（冲稳保梯队建模）**
   - 严格遵循平行志愿制度，模型调用外部独立短代码执行匹配：
     - **捡漏 (前5组) **：向位次差极大的好学校的绝对冷门发牌。
     - **冲段 (5组) **：上摸 20%~5%。
     - **稳段 (15组)**：上下浮动匹配。
     - **保段 (15组)**：下压30%的绝对托底压舱石策略。

4. **🗣️ 极度贴地飞行的“张老炮”风格输出**
   - “有矿去读中外合作、没背景别硬填法学管理”
   - “只要上了平行志愿，不勾服从调剂就是找退档，你是想全家陪绑吗？”

## 🚀 如何安装和运行此 Skill？

由于这是一个为大语言模型打造的系统级 Prompt（包含了严谨的约束性数据交互框架），您需要在一个支持外部目录查表的 Agent 环境（如具备 MCP 框架的 Claude/Cursor 等客户端）下使用它：

1. **装载 Skill 提示词：** 
   将本项目根目录下的 `SKILL.md` 的内容，粘贴/设定为您的 Agent System Prompt（系统角色设定）。

2. **装载硬性录取数据：**
   此技能需要读取数据表才能不胡说八道！请在 `data/` 目录下放置当下的三份核心基准数据：
   - `2023-2025各专业组录取最低分.xlsx`
   - `2023-2025逐分段统计表.xlsx`
   - `江苏2025年填报志愿参考表.xlsx`

3. **激活口令：**
   系统准备就绪后，对大模型输入关键词：`高考志愿`、`能不能上`、`报志愿`，系统就会瞬间带上前文提到的认知和脾气，开始对您盘查 10 大硬性基础信息。

模型指令中已严格加入了禁止性红线，包括：
- **绝对不**直接向您输出没有组别代码只有学校名的废话（江苏看专业组！）。
- **绝对不**在表格里输出被拼接过的混杂长代码（精确切割独立短代码）。
- **绝对不**编造不客观的分数推演。
